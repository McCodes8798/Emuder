# Descendants Tate Emmanuel Muderhwa

A family tree website showing the descendants of Tate Emmanuel Muderhwa.

## Features

- Interactive family tree with 3 generations
- Click on any person to see their details and children
- Search functionality to quickly find family members
- Mobile-friendly responsive design
- Location display for family members

## Setup

No installation required - just open `index.html` in a web browser.

## Deployment

This site can be deployed to:
- Netlify (drag and drop)
- GitHub Pages
- Vercel
- Any static hosting service

Simply upload all files maintaining the folder structure.
