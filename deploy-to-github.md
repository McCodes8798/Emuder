# Deploy to GitHub Pages

## Quick Steps:

1. Go to https://github.com and sign up/login
2. Click the "+" icon in top right, select "New repository"
3. Name it: `muderhwa-family-tree`
4. Make it **Public**
5. Click "Create repository"

## Upload Files:

### Option A: Using GitHub Website (Easiest)
1. Click "uploading an existing file"
2. Drag ALL your files from the murderhwa_desc folder:
   - index.html
   - family-data.js
   - script.js
   - styles.css
   - README.md
   - DEPLOYMENT.md
   - The entire `images` folder with all images
3. Click "Commit changes"

### Option B: Using GitHub Desktop
1. Download GitHub Desktop from https://desktop.github.com
2. File > Add Local Repository
3. Point to your murderhwa_desc folder
4. Commit and Push

## Enable GitHub Pages:

1. Go to your repository Settings
2. Scroll to "Pages" section
3. Under "Source" select "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Click Save

## Get Your URL:

After 2-3 minutes, your site will be at:
`https://YOUR-USERNAME.github.io/muderhwa-family-tree`

Share that URL with your family!

## Update Later:

To make changes:
1. Edit files locally
2. Upload to GitHub again
3. Changes appear automatically in 1-2 minutes
