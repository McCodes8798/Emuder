# How to Deploy and Share Your Family Tree

## Option 1: Netlify Drop (Easiest - 30 seconds)

1. Go to https://app.netlify.com/drop
2. Drag your entire project folder onto the page
3. You'll get a URL like `https://random-name-123.netlify.app`
4. Share that URL with your family!

**Pros:** Super quick, no signup needed, free
**Cons:** Can't update it easily later

## Option 2: GitHub Pages (Best for updates)

1. Create a GitHub account at https://github.com
2. Create a new repository called "family-tree"
3. Upload all your files to GitHub
4. Go to Settings > Pages
5. Select "main" branch as source
6. You'll get a URL like `https://yourusername.github.io/family-tree`
7. Share the URL!

**Pros:** Easy updates, version control, free
**Cons:** Need to set up GitHub account

## Option 3: Vercel (Also very easy)

1. Go to https://vercel.com
2. Sign up with GitHub or email
3. Click "Add New Project"
4. Drag and drop your project folder
5. Get instant URL to share!

**Pros:** Very fast, automatic deployments, free
**Cons:** Need to sign up

## Local Testing

Your site is currently running on:
- **Your computer:** http://localhost:8000

To test before sharing, just open that URL in your browser!

## What to Share

Once deployed, share:
1. The URL
2. Tell them to click on any person to see more info
3. They can search by name
4. Works on phones too!
